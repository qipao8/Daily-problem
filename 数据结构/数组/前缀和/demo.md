## 前缀和（类似高中的数列求和）

```
let nums=[3,2,3,2,1,9,8,6,2,2,7,5,9,7,2,6,3,5,1,1,3,8,8,10,7,1,8,9,3,4,1,9,1,10,4,4,7,6,9,9,4,3,2,7,7,7,1,3,9,4,9,4,7,3,7,8,2,5,4,6,2,0,7,3,6,3,2,4,4,1,8,9,2,3,9,7,2,3,9,3,3,8,2,4,9,7]
let Sum = new Array(nums.length + 1).fill(0)
Sum.forEach((v, i) => i && (Sum[i] = Sum[i - 1] + nums[i - 1]))
```
前缀和数组,长度为nums.length+1，首项为0，数组前i项的和为Sum[i-1]
